# 🌍 Tripadvisor Clone

A responsive **Tripadvisor Clone** built using **HTML5** and **CSS3**. This project recreates the modern Tripadvisor homepage with a clean user interface, responsive layout, and attractive travel destination sections.

---

## 📌 Features

- 🏠 Modern Homepage Design
- 📱 Fully Responsive Layout
- 🔍 Search Bar
- 🧭 Navigation Menu
- 🌎 Popular Destinations
- 🏨 Hotels Section
- 🍽️ Restaurants Section
- ✈️ Things to Do Section
- 📸 Beautiful Travel Images
- 🎨 Clean UI Design
- ⚡ Fast Loading Static Website

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome Icons
